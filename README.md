# E2 Submesh Materials
## Installation instructions:
1. Download and extract the .zip (Code -> Download ZIP).
2. Navigate into the `E2-Submesh-Materials-main` folder. You should see two subfolders: `addons` and `data`.
3. Merge the `addons` folder with your `garrysmod/addons` folder.
4. Merge the `data` folder with your `garrysmod/data` folder.
