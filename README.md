# E2 Submesh Materials
**Disclaimer for server owners:** DO NOT INSTALL THIS ON A SERVER.
This E2 extension is designed for **singleplayer use only** and has not been tested in multiplayer. The function it uses is expensive and may cause performance issues (read: lag/crashes) in multiplayer, since E2 is serverside.

That said, this shouldn't cause any problems while *playing* on a server - I've been doing this for months with no issue.
## Installation instructions:
1. Download and extract the .zip (Code -> Download ZIP).
2. Navigate into the `E2-Submesh-Materials-main` folder. You should see two subfolders: `addons` and `data`.
3. Merge the `addons` folder with your `garrysmod/addons` folder.
4. Merge the `data` folder with your `garrysmod/data` folder.
